Projetos

HypScrum

1º Semestre

O HypScrum é uma aplicação web educacional voltada para o ensino do framework Scrum. Ela permite que usuários acessem conteúdos instrutivos, realizem avaliações e recebam certificados de conclusão. O sistema foi desenvolvido com HTML, CSS, JavaScript e PostgreSQL, com prototipação feita no Figma e acompanhamento via Trello.

HypoHeat

2º Semestre

O HypoHeat é uma aplicação web interativa que permite a visualização e análise de dados sobre queimadas no Brasil, como área queimada, risco de fogo e focos de calor. Os dados são obtidos diretamente do banco de dados BDQueimadas, do INPE. A aplicação inclui visualizações por estado e bioma, com mapas e gráficos interativos.
