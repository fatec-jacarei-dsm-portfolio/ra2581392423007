Projetos Acadêmicos

HypScrum

1º Semestre

O HypScrum é uma aplicação web educacional voltada para o ensino do framework Scrum. Ela permite que usuários acessem conteúdos instrutivos, realizem avaliações e recebam certificados de conclusão. O sistema foi desenvolvido com HTML, CSS, JavaScript e PostgreSQL, com prototipação feita no Figma e acompanhamento via Trello.

Participação: Contribuí na prototipagem das telas de login e cadastro e no desenvolvimento do front-end do sistema. Trabalhei na componetização do header com controle de visibilidade, além de implementar as telas de início e capítulos, considerando os estados de usuário logado e deslogado. Também desenvolvi a tela de conteúdo sobre Eventos do Scrum e participei da estilização em CSS, garantindo uma interface mais organizada e intuitiva.

HypoHeat

2º Semestre

O HypoHeat é uma aplicação web interativa que permite a visualização e análise de dados sobre queimadas no Brasil, como área queimada, risco de fogo e focos de calor. Os dados são obtidos diretamente do banco de dados BDQueimadas, do INPE. A aplicação inclui visualizações por estado e bioma, com mapas e gráficos interativos. O sistema foi desenvolvimento com React, Typescript, PostgreSQL e PostGis para podermos trabalhar com os dados.

Participação: Participei da elaboração de diagramas de sequência para representar o fluxo do sistema, desenvolvi uma stored procedure para importar dados do DBQueimadas e colaborei na implementação dos gráficos, contribuindo para a análise visual das informações. Essas atividades integraram teoria e prática, agregando valor ao projeto e fortalecendo meu aprendizado.

HypoOrbit

3º Semestre

HypoOrbit é uma aplicação web para visualização e comparação de dados geoespaciais gratuitos provenientes de diferentes satélites. A plataforma centraliza informações que hoje estão dispersas, permitindo que usuários identifiquem de forma simples quais satélites possuem dados para uma área de interesse, consultando variáveis geoespaciais como NDVI, EVI, temperatura da superfície e umidade do solo. O sistema foi desenvolvimento com React, Typescript e MongoDB.

Participação: Contribuí no desenvolvimento realizando o consumo da STAC API, estudando os serviços geoespaciais disponíveis e implementando a listagem dinâmica dos satélites, incluindo a rota para consultar dados por localização. Também integrei o serviço de filtro com a interface, garantindo o funcionamento completo do fluxo de consulta. Fui responsável pela configuração do sistema em Docker, criando o ambiente de execução com docker-compose, Dockerfile, .env e banco de dados, além de executar testes, ajustes e finalização do ambiente.
